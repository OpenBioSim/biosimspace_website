BioSimSpace.Convert.to

BioSimSpace.Convert.to(obj, format='biosimspace', property_map={})[source]

Convert an object to a specified format.

Parameters:

  • obj The input object to convert.

  • format (str) The format to convert to.

  • property_map (dict) A dictionary that maps system properties to their user defined values. This allows the user to refer to properties with their own naming scheme, e.g. { charge : my-charge }

Returns:

The object in the converted format.

Return type:

converted_obj
